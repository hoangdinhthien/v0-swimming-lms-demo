- cần phải cập nhật request body lại api createClass
- đây là request body của createClass NẾU NHƯ TẠO 1 LỚP HỌC:
[
    {
        "course": "69079ef9c5d0d1fc7a75e5bb",
        "name": "test 6",
        "instructor": "683c810c0707d1bbac3063f0",
        "show_on_regist_course": false
    },
    {
        "course": "69079ef9c5d0d1fc7a75e5bb",
        "name": "test 7",
        "instructor": "683c810c0707d1bbac3063f0",
        "show_on_regist_course": false
    }
]
- đây là request body của createClass NẾU NHƯ TẠO NHIỀU LỚP HỌC CÙNG LÚC:
{
    "course": "69079ef9c5d0d1fc7a75e5bb",
    "name": "test 6",
    "instructor": "683c810c0707d1bbac3063f0",
    "show_on_regist_course": false
}
- về cơ bản 2 request body này chỉ khác nhau ở việc khi tạo nhiều lớp học cùng 1 lúc thì request body sẽ là 1 array objects
